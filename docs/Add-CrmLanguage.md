# Add-CrmLanguage

## SYNOPSIS
Provision a language.

## SYNTAX

```
Add-CrmLanguage [-LocaleId] <Int32> [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Provision a language.

## EXAMPLES

### Example 1
```
PS C:\> Add-CrmLanguage -LocaleId 1043
```

Provision the Dutch language.
